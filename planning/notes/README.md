# Planning Notes

This folder contains quick notes, ideas, and brainstorms for aipedia.org.

## How to Use
- Create a new file for each topic/idea
- Use format: `YYYY-MM-DD-topic-name.md`
- Notes can be rough - they're for capturing ideas quickly
- Promote mature ideas to proper ADRs or roadmap items

## Note Template

```markdown
# [Topic/Idea Name]

**Date**: YYYY-MM-DD
**Author**: [Your name]

## Idea
Brief description of the idea.

## Why
Why is this worth considering?

## How
Rough thoughts on implementation.

## Questions
- Open question 1?
- Open question 2?

## Next Steps
- [ ] Action item 1
- [ ] Action item 2
```
